# Overview
- This was a project used to mimic a Borrowing System used in Libraries for books. 
- This app was built on Python libraries: tKinter, SQLAlchemy

# Features
- Membership
    - Creation 
    - Deletion 
    - Update
    
- Books 
    - Acquisition 
    - Withdrawal

- Loans
    - Book Borrowing 
    - Book Return
    
- Reservations
    - Make a Reservation 
    - Cancel a Reservation
    
- Fines
    - Fine Payment

- Reports
    - Search
    - Books on Loan
    - Books on Reservation
    - Outstanding Fines
    - Books on Loan to Member
